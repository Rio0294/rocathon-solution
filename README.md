# rocathon-solution
Build a creator search engine for rocathon where search creators by category and vibe.Then rank them by GMV and projected score. 

The submission returns 8 results instead of 10. A `match_threshold: 0.3` was chosen to ensure only creators with genuine semantic relevance to the query are returned, balancing creative fit with commercial viability. Lowering the threshold would pad results with loosely-related creators, undermining the hybrid scoring philosophy.
